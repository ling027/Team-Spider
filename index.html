Project:IMDB-ish Website

Project Description: Building a website mimicking IMDB, where we will recreate the look and functionalities of the IMDB website, core features include: advanced search, browsing, user accounts, rating, reviewing, and personalized watchlist. Our webiste should allow users to create account, sign in, interact with the contents to leave ratings and reviews, and add films, tv shows, and video game to their watchlist if they were interested.
